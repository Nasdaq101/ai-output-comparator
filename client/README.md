# Frontend - React + Vite

React frontend for AI Comparator application.

## Tech Stack

- React 19.2.0
- Vite 7.2.2
- CSS3

## Development

```bash
npm install
npm run dev
```

Frontend runs at: http://localhost:5173

## Build

```bash
npm run build
```

## Structure

```
src/
├── App.jsx      # Main component
├── App.css      # Styles
├── main.jsx     # Entry point
└── index.css    # Global styles
```
